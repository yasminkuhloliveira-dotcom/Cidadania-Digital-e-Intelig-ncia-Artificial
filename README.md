# Portal Conexão Segura 🌐🤖
> Projeto prático de conscientização sobre **Cidadania Digital e Inteligência Artificial**, com foco nos impactos das Deepfakes e da Desinformação na Sociedade moderna.

Este portal foi desenvolvido como parte das atividades avaliativas de Tecnologia de 2026. O objetivo principal é educar a comunidade interna e externa a identificar mídias sintéticas manipuladas de forma maliciosa por algoritmos de IA, promovendo uma navegação digital crítica, segura e ética.

## 🚀 Funcionalidades Principais
* **Conteúdo Autoral Educativo:** Explicações claras e diretas sobre os riscos da desinformação estruturada por IA e guias visuais/textuais para identificar adulterações.
* **Acessibilidade (Modo Escuro):** Botão dinâmico que altera o contraste do site, respeitando as preferências salvas no navegador do usuário.
* **Quiz Interativo:** Script dinâmico que valida respostas do usuário em tempo real, fornecendo feedbacks pedagógicos baseados na escolha informada.
* **Design Responsivo:** Layout totalmente adaptável para dispositivos móveis (smartphones e tablets) utilizando CSS Flexbox e Media Queries.

## 🛠️ Tecnologias Utilizadas
* **HTML5:** Estruturação semântica avançada (`<main>`, `<section>`, `<form>`).
* **CSS3:** Estilização baseada em variáveis de escopo (`:root`), efeitos hover e design flexível via Flexbox.
* **JavaScript (ES6):** Manipulação dinâmica de elementos do DOM, escuta de eventos e persistência local (`localStorage`).

## 🤖 Uso de Inteligência Artificial no Desenvolvimento
Em conformidade com os critérios éticos do projeto, documentamos abaixo os prompts utilizados para a co-criação da estrutura lógica deste código:
1. *Prompt para Engenharia de Software:* "Gere uma estrutura de HTML5 semântico e um script JS puro que faça a validação de um formulário de quiz sem dar refresh na página, exibindo caixas de feedback com classes dinâmicas de erro e sucesso."
2. *Prompt para Design de Interface:* "Construa uma folha de estilos CSS focada em segurança digital utilizando uma paleta baseada em tons de azul tecnológico, cinza e branco, com suporte nativo a light/dark mode baseado em atributo data-theme."
